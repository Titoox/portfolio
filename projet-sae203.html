01 — Contexte

Le projet

SAE203 est un projet universitaire BUT MMI consistant à développer une application web complète en PHP/MySQL. L'objectif : créer un répertoire de marques de mode avec une interface publique de consultation et un espace admin de gestion.

Le projet couvre l'ensemble du cycle de développement — conception de la base de données, développement back-end en PHP procédural, sécurité, et déploiement sur VPS Linux avec configuration Apache.

  • 1
    Interface publique Listing des marques en grille, fiche détail, recherche par nom.
  • 2
    Espace admin CRUD Ajouter, modifier, supprimer des marques et collections via formulaires sécurisés.
  • 3
    Upload d'images Gestion des photos de marques avec validation et stockage serveur.
  • 4
    Déploiement VPS Mise en ligne sur serveur Linux — Apache, MariaDB, SSH, VirtualHost.
02 — Stack technique

Les technologies

PHP
Backend procédural — traitement des formulaires, requêtes PDO préparées, upload de fichiers, protection XSS.
PDO · htmlspecialchars · move_uploaded_file · strip_tags
MySQL
2 tables liées — marques et collections. Requêtes préparées, recherche avec LIKE, jointures entre tables.
SELECT · INSERT · UPDATE · DELETE · LIKE · JOIN
HTML / CSS
Interface responsive sans framework — grille de cartes, fiches produits, formulaires admin. Déployé sur VPS.
Flexbox · Grid · Responsive · Apache · VirtualHost
